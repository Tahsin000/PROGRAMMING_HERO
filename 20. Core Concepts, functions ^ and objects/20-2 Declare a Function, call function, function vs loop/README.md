# **Declare a Function, call function, function vs loopPreviousNext**

## function.js

```jsx
function startFan() {
  console.log("Stand up");
  console.log("walk towards the switch");
  console.log("press the switch");
}

// call the function
startFan();
```

## parameter.js

```jsx
function bringSingara(money) {
  console.log("mama singara den");
  console.log(money);
}
bringSingara(100);
```
