Difference between map filter forEach and find?

map(), filter(), forEach(), and find() are all array methods in JavaScript that can be used to iterate through an array and perform operations on its elements. However, they have different functionalities and return values

forEach()

forEach() is used to loop through each element in an array and perform some operation or action on it. It does not return a new array.

const numbers = [1, 2, 3];
numbers.forEach(number => console.log(number));
// Output: 1 2 3

map()

map() is used to loop through an array and perform some operation on each element of the array. It returns a new array with the modified elements.

const numbers = [1, 2, 3];
const squares = numbers.map(number => number * number);
console.log(squares);
// Output: [1, 4, 9]

filter()

filter() is used to loop through an array and create a new array with elements that meet a certain condition. It returns a new array with the filtered elements.

const numbers = [1, 2, 3, 4, 5];
const evenNumbers = numbers.filter(number => number % 2 === 0);
console.log(evenNumbers);
// Output: [2, 4]

find()

find() is used to loop through an array and return the first element that meets a certain condition. It returns the first matching element or undefined if no matching element is found.

const numbers = [1, 2, 3, 4, 5];
const firstEvenNumber = numbers.find(number => number % 2 === 0);
console.log(firstEvenNumber);
// Output: 2
