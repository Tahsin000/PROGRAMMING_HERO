What does the quizTimer(), chooseQuiz() function do?

quizTimer()

The quizTimer function sets up a timer that increments every second and displays the elapsed time on the webpage.

If the dismiss argument is true, the function clears the interval that is set by the timer and stops the timer.

Otherwise, the function sets up an interval of one second using the setInterval function. Inside the interval, the function calculates the elapsed time by dividing the count by 60 to get the number of minutes and using the remainder operator to get the number of seconds. It then formats the minutes and seconds values to ensure that they are always two digits long.

chooseQuiz()

The function takes two arguments: index and givenAns. The index argument represents the index of the quiz question in the quizData array, while givenAns represents the answer given by the user for that question.

The function first checks if an answer for the given question already exists in the answers array by calling the find method on the answers array and searching for an answer with an id property that matches the id property of the quiz question at the specified index in the quizData array.
